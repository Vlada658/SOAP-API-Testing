# SOAP-API-Testing
**Описание задания:**  
Создана коллекция в Postman для тестирования SOAP-сервиса получения информации о странах.

**Что было сделано:**  
- Изучена WSDL-документация SOAP-сервиса  
- Настроены SOAP-запросы в Postman  
- Протестированы методы:
  - ListOfContinentsByName  
  - CountryName  
  - FullCountryInfo  
  - LanguageName  

**Ссылка на сервис:**
http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso?WSDL
